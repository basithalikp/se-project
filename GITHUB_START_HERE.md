# 🚀 GitHub + Cloud Hosting - START HERE

## ⚡ What You Need to Do (4 Simple Steps)

### Step 1: Create GitHub Repo (2 min)
```
1. Go to https://github.com/new
2. Name: smart-water-system
3. Visibility: Public
4. Create
```

### Step 2: Push Code to GitHub (5 min)
```powershell
cd d:\SmartWaterSystem
git init
git add .
git commit -m "Initial commit: Smart Water System"
git remote add origin https://github.com/YOUR_USERNAME/smart-water-system.git
git branch -M main
git push -u origin main
```

**⚠️ Replace YOUR_USERNAME with your GitHub username**

### Step 3: Deploy Frontend to Vercel (5 min)
```
1. Go to https://vercel.com
2. Sign in with GitHub
3. Import smart-water-system repo
4. Deploy
```

**Live Frontend:** https://smart-water-system.vercel.app

### Step 4: Deploy Backend to Railway (5 min)
```
1. Go to https://railway.app
2. Sign in with GitHub
3. Create project
4. Select smart-water-system repo
5. Set env variables:
   - MONGODB_URI = your MongoDB URL
   - JWT_SECRET = random secret
6. Deploy
```

**Live Backend:** https://your-backend.railway.app
