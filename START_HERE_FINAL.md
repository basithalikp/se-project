### Customize the Theme

**Change Colors:**
```css
:root {
  --primary-light: #f59e0b;  /* Change from green to amber */
}
```

**Adjust Blur:**
```css
backdrop-filter: blur(8px) saturate(180%);  /* Less blur */
backdrop-filter: blur(25px) saturate(180%); /* More blur */
```

**Brighten Text:**
```css
color: rgba(255, 255, 255, 0.95);  /* More opaque */
```
